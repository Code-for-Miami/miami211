#Miami 211 Rebuild#

A Sinatra app for the [Ohana API][]

[Ohana API]:https://github.com/codeforamerica/ohana-api

###Under the hood:
* Ruby 2.3.0
* Sinatra
* Sinatra Reloader
* Ohanakapa

###Quick start:

    # Initial setup:
    cd [project_directory]
    bundle install 
    
    # Start the app
    cd [project_directory]
    ruby server.rb
    
    # In the browser:
    localhost:4567
