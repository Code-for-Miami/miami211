#Miami 211 Rebuild#

A Sinatra app for the [Ohana API][]

[Ohana API]:https://github.com/codeforamerica/ohana-api

##Under the hood
* Ruby (tested up to v2.3.0)
* Sinatra
* Ohanakapa
* jQuery

##Quick Start
###Initial setup

    cd [project_directory]
    bundle install 
    
###Starting the app

    cd [project_directory]
    ruby server.rb
    
###In the browser

    localhost:4567
